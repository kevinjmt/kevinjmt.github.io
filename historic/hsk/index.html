DESCRIPTION

Le Hanyu Shuiping Kaoshi (HSK ; chinois : 汉语水平考试 ; pinyin : Hànyǔ Shuǐpíng Kǎoshì), traduit par le test de compétence en chinois, est le test standardisé de compétence en chinois standard (un type de chinois mandarin) de la Chine continentale pour les locuteurs non natifs tels que les étudiants étrangers et les Chinois d'outre-mer. Le test est administré par le Hanban, une agence du ministère de l'éducation de la République populaire de Chine.

(Traduit de l'anglais)
